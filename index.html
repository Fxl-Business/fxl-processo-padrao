Como Evoluir Este Processo

Este repositório é evoluído via Claude Project "FXL — Processo Padrão". Você discute a melhoria lá, recebe um prompt estruturado, cola no Claude Code dentro do repositório, e faz o commit. O GitHub Pages atualiza automaticamente.
