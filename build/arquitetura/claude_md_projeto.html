Como usar

  1. Copie o template abaixo
  2. Cole como CLAUDE.md na raiz do repositório do projeto
  3. Substitua todos os campos [PREENCHER]
  4. Substitua [CAMINHO_ABSOLUTO] pelo caminho real na sua máquina
  5. Atualize a seção "Sprints" conforme o projeto avança

Nota sobre caminho absoluto

⚠️ Substitua [CAMINHO_ABSOLUTO] pelo caminho real na sua máquina antes de usar o CLAUDE.md em qualquer projeto. O Claude Code não consegue inferir o caminho automaticamente.

Exemplo: se os seus projetos ficam em /Users/cauet/projetos, o caminho completo seria /Users/cauet/projetos/fxl-processo-padrao/build/arquitetura/premissas_gerais.md

Esta abordagem por caminho absoluto funciona porque há um único operador na máquina. Para escalar no futuro (múltiplos operadores ou CI/CD), migrar para git submodule ou variável de ambiente configurada por operador.
