# Biblioteca de KPIs — Referência Operacional

## Objetivo

Fornecer repertório de KPIs por módulo para apoiar a Fase 1 e reduzir lacunas de escopo.

---

## Regra de uso

- Usar como apoio, não como checklist fechado
- Confirmar coluna de dados antes de considerar um KPI como aprovado
- Sinalizar KPIs sem base de dados como bloqueados

---

## Módulos base

### Comercial
- Faturamento
- Ticket médio
- Conversão
- Vendas por vendedor

### Operacional
- Lead time
- Volume por etapa
- SLA
- Produtividade por equipe

### Financeiro
- Receita
- Despesa
- Margem
- Inadimplência

### Inputs
- Volume importado
- Taxa de erro
- Tempo até atualização

---

## Critério de conclusão

O uso desta biblioteca está correto quando o Briefing diferencia:
- KPIs confirmados
- KPIs sugeridos
- KPIs bloqueados por ausência de dados
